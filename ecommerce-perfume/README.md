# README for E-commerce Perfume Business

## Project Overview
This project is an e-commerce website dedicated to selling perfumes. It features a user-friendly interface that allows customers to browse and purchase a variety of fragrances.

## Project Structure
```
ecommerce-perfume
├── src
│   ├── index.html        # HTML structure for the homepage
│   ├── styles
│   │   └── styles.css    # CSS styles for the homepage
│   └── scripts
│       └── main.js       # JavaScript for interactivity
├── package.json          # NPM configuration file
└── README.md             # Project documentation
```

## Features
- **Homepage**: Displays featured perfumes and categories.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Interactive Elements**: Users can add products to their cart and filter selections.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd ecommerce-perfume
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your browser to view the homepage.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js (for package management)
